This is the help page for LutherCourseProposal.

Creating a New Proposal

  1. The first step is to click the New Course button next to the My Changes section.
  2. You will be then taken to the fill-out page which contains all of the neccessary and optional fields for a new course. The necessary fields are the ones marked with a red asterisk.
  3. Fill out the form as needed.
  4. Once finished, hit the submit button at the bottom of the page.

Editing a Proposal

  1. The first step to editing course is under the My Changes section click the pencil icon next to the course name.
  2. You will be taken to the course-edit form which is the same as the course-proposal form just with your previous information filled in.
  3. Make your edits.
  4. Once finished, click the submit button at the bottom of the page.
Course Title
This is the title of the course. It should be descriptive. (Required)
Department
This is the department for the course. It is a drop-down list of of departments. (Required)
Course Number
This is the number for the course. (Required)
Instructor(s)
These are the instructors for the course. This consists of a drop-down menu and then the result must be added using the add button. (Required)
General Requirements Satisfied
These are the general requirements for a course. This is done using the same way as Instructor(s).
Credit Hours
Number of credit hours. (Required)
Estimated Enrollment
Estimated number of students who probably will enroll. (Required)
Capacity
This is the maximum capacity for a course. (Required)
Terms Offered
This is specifying the terms that the course is offered. It is a drop-down list that can have multiple items by ctrl-clicking each item. To remove is another ctrl-click. (Required)
Prerequisites
These are the prerequisites for a course.
Fees
This is for any additional fees necessary for the course.
Catalog Description
This is the description of the course that will go into the catalog. (Required)
Rationale
This is the reasoning behind the course and its existance. (Required)
Impact on Major/Minor/Program
This is for describing the ambitions of a course upon completion.
Staffing Implications
This is for describing the neccessary staff for the course.
